Step 1: Firebase Setup

We use Google Firebase to store data securely for free.

1 Create Project

Go to the Firebase Console and click Add Project. Name it anything (e.g., "MyBudgetApp") and disable Google Analytics for simpler setup.

2 Enable Authentication

On the left sidebar, click Build > Authentication.

  • Click Get Started.
  • Select Google from the Sign-in providers list.
  • Toggle the switch to Enable.
  • Select your support email and click Save.
3 Enable Database

On the left sidebar, click Build > Firestore Database.

  • Click Create Database.
  • Choose Start in Production Mode.
  • Select a location close to you (e.g., asia-south1 for India) and click Enable.
4 Set Security Rules

Go to the Rules tab inside Firestore Database and replace the code with this (copy-paste):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      // Security: Only allow users to read/write their OWN data
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Click Publish.

Step 4: Domain Whitelist (CRITICAL) ‚ö†Ô∏è

STOP! Do not skip this.
If you do not do this step, Google Login will fail with an error: auth/unauthorized-domain.

Google blocks sign-ins from unknown websites for security. You must authorize your website domain.

! How to Whitelist
  1. Go to Firebase Console > Authentication > Settings tab.
  2. Click on Authorized Domains.
  3. Click Add Domain.
  4. Enter your live website URL (e.g., budget-app.netlify.app or syedsubhani.in).
  5. Click Add.

(Note: `localhost` is allowed by default for testing on your computer).

Step 5: PWA Customization

To make the app installable on phones with your own branding:

  1. Icons: Replace the images in the icons/ folder. Keep the filenames and sizes (192x192, 512x512) exactly the same.
  2. Name: Open manifest.json and change the "name" and "short_name" fields to your app name.
  3. Theme Color: Open `manifest.json` and change "theme_color" to match your brand.

FAQ & Admin Panel

Q: Where is the Admin Panel?

A: It's built into Google! Instead of a basic custom panel, this app uses the enterprise-grade Firebase Console as its backend admin suite.

  • View Users: Go to the Authentication tab to see all signed-up users. You can disable or delete accounts there.
  • Manage Data: Go to the Firestore Database tab to view, edit, or delete any user's financial records in real-time.
  • Analytics: Enable Google Analytics in the console to track daily active users and retention.
Q: Can I change the currency to USD?

A: Yes. Open app.js and search for the function formatMoney. Change en-IN to en-US and replace the ‚Çπ symbol with $.

Q: How do I reset all data?

A: Use the "Clear All Data" button in the app's Profile tab settings. This wipes local storage. To wipe cloud data, delete the user document in Firebase Console.
