Thank you for purchasing The Ultimate Gym 2.0! This guide will help you launch your own Gym Management SaaS in minutes.

✅ Serverless: This app runs entirely on Google Firebase (Free Tier). You do not need PHP, MySQL, or expensive hosting.

Step 1: Firebase Setup

We use Google Firebase to store data securely. Follow these steps to create your free database.

1

Create Project

Go to the Firebase Console and click Add Project. Name it (e.g., "MyGymApp") and disable Google Analytics for a simpler setup.

2

Enable Authentication

On the left sidebar, go to Build > Authentication.

  • Click Get Started.
  • Select Google from the Sign-in providers list.
  • Toggle the switch to Enable.
  • Select your support email and click Save.
3

Create Database

On the left sidebar, go to Build > Firestore Database.

  • Click Create Database.
  • Choose Start in Production Mode.
  • Select a location close to you (e.g., asia-south1 for India) and click Enable.
4

Set Security Rules

Go to the Rules tab inside Firestore Database. Delete everything and paste this code. This ensures each gym owner can only see THEIR own data.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Allow users to read/write only their own gym data
    match /gyms/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
    
  }
}

Click Publish.

Step 2: Connect the App

Now link your code to the database you just created.

  1. In Firebase Console, click the Gear Icon ⚙️ (Project Settings).
  2. Scroll down to "Your Apps" and click the Web ( </> ) icon.
  3. Register the app (enter any nickname).
  4. You will see a firebaseConfig code block. Copy only the values inside the braces.
  5. Open the file firebase-init.js in your source code folder.
  6. Paste your keys into the configuration section.
// firebase-init.js
const firebaseConfig = {
    apiKey: "AIzaSyD-Your-API-Key-Here",
    authDomain: "your-project.firebaseapp.com",
    projectId: "your-project-id",
    storageBucket: "your-project.appspot.com",
    messagingSenderId: "123456789",
    appId: "1:123456789:web:abc..."
};

Step 3: Domain Whitelist (CRITICAL) ⚠️

STOP! If you skip this, Google Login will fail with an error: auth/unauthorized-domain.

Google blocks sign-ins from unknown websites for security. You must authorize your website domain.

  1. Go to Firebase Console > Authentication > Settings tab.
  2. Click on Authorized Domains.
  3. Click Add Domain.
  4. Enter your live website URL (e.g., my-gym-app.netlify.app or syedsubhani.in).
  5. Click Add.

(Note: localhost is allowed by default for testing).

Managing Members

Click the + New Member button on the dashboard to add clients.

  • Photo: Upload a photo for easier identification.
  • Plan: Select duration (1 Month to 1 Year). Expiry is calculated automatically.
  • Phone: Enter a valid 10-digit number for WhatsApp integration.

Attendance System

Track daily gym visits.

  • Mark In: Click the gray icon. It turns Green.
  • Revoke: Click the green icon again to remove attendance for today (undo).
  • History: Click the Clock icon to see the full list of dates attended.

Fitness & Workouts

Navigate to the Fitness / BMI tab.

  • BMI Tracking: Enter Weight (kg). Height is set once during registration. The system tracks weight loss/gain progress.
  • Workout Plan: Use the text area to assign a weekly schedule (e.g., "Mon: Chest, Tue: Back"). This saves to the member's profile.

Finance & Billing

The app automatically tracks income when you add members or renew plans.

  • Manual Entry: Use "Add Entry" for expenses like Rent or Electricity.
  • Invoices: Click the icon to generate a PDF receipt.
  • Export: Use the Data IO tab to download CSV reports for Excel.

FAQ & Troubleshooting

Is my data safe?

Yes. Data is stored in Google Cloud. Only the logged-in owner can access their specific gym's data due to the Security Rules we set up in Step 1.

How do I reset the app?

To clear local cache, simply log out and log back in. To delete data permanently, go to your Firebase Console > Firestore and delete the `gyms` collection.

Can I use this offline?

This is a cloud-based app. You need an internet connection to load and save data securely.
